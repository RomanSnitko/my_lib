# my_lib
Реализовал собственную C++ библиотеку на базе CMake, настроил сборку с использованием target-based подхода. Сделал библиотеку устанавливаемой и релоцируемой (install/export/config package), подключил её во втором проекте через find_package как внешний пакет. Добавил генерацию файлов и кастомные таргеты, интегрировал GoogleTest для unit-тестирования. Обернул библиотеку в Conan-пакет, добавил внешнюю зависимость (zlib) и настроил использование библиотеки в потребителе через Conan.
